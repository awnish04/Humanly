1. Dashboard calls `/api/admin/health` every time it refreshes
2. Health endpoint tests all 5 services in parallel
3. Each service check measures response time
4. Results show in the System Status chart with:

   - Latency (ms)
   - Status (OK/Error)
   - Overall system health indicator
